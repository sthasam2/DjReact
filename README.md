# DjReact
A blog app using **Django** _(Django Rest Framework)_ and the popular **React.js** Library.

## Dependencies
* ### Frontend
    Node, npm, react, axios, react-router-dom, antd

* ### Backend
    Python, django, django-rest-framework, django-cors-headers
